Smart Home HCI Research

Our smart home, MARS, is an environment based on machine learning algorithms to observe user behaviors and offer real-time services through IoT devices. Due to MARS cannot predict user behaviors perfectly and users need to know the learning state of MARS to interact with it; thus, an interface for the users to understand MARS is indispensable. To find the most fitting choice among various user interfaces for MARS in a fallible environment, I am now designing the experiment process through famous methods in ubiquitous computing.

Methodologies: Speed Dating, Wizard of Oz and User Enactment

Auto Tracking System Realized on the Pan-Tilt Platform

Researching on the automatic tracking system based on Pan-Tilt platform and realizing the PID control on Servo and the tracking algorithms, such as the three-step search, diamond search, etc. Additionally, we compared different variations like RGB, HSV, and Grayscale and added other features to improve the efficiency, HOG for example.

Technologies: C/C++, OpenCV, Servo and Control Engineering

Watershed Segmentation and GUI Toolkits

This project is the final project in the image processing course. The purpose is to develop the Watershed Segmentation algorithm to segment every cell from a biopsy image and provide GUI toolkits for professional people to navigate. I added some user-friendly functions so that the users could magnify and collect/save cell information simply by clicking the mouse. As a result, the users could navigate the image and record cell features efficiently.

Technologies: C/C++, OpenCV
